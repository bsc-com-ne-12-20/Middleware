API Documentation

Core Authentication

Endpoint Method Description
/api/logout/ POST Invalidate auth token (logout agent).
/api/change-password/ POST Change password for authenticated agent.
/api/profile/ GET Get current authenticated agent's profile.
/api/get-balance/ POST Get agent balance by email.
/api/get-username/ POST Get agent username by email.
/api/agent-balance/ GET Get agent balance by agent code.
/api/get-agent-username/ POST Get agent username by agent code.

Financial Transactions

Deposits

Endpoint Method Description
/api/v1/dpst/ POST Create a new deposit.
/api/v1/dpst/dpst-hstr/ GET Get deposit history filtered by agent code and time range (day, week, month).
/api/v1/dpst/get-balance/ GET Get current balance for an agent by agent code.
/api/v1/dpst/analytics/ GET Get analytics summary and growth metrics for deposits, withdrawals, and transfers.

Transfers

Endpoint Method Description
/api/v1/agent-trsf/ POST Create a new transfer.
/api/v1/agent-trsf/trsf-hstr/ GET Get transfer history filtered by agent code and time range (day, week, month).

User Withdrawals

Endpoint Method Description
/api/v1/wtdr/ POST Create a new withdrawal request to an agent.
/api/v1/wtdr/wdr-to-agent-hstr/ GET Get withdrawal history filtered by agent code and time range (day, week, month).
/api/v1/wtdr/dpst/ POST Create a deposit record related to withdrawals.

Agent Management

Endpoint Method Description
/api/v1/agent-usr/ GET, POST List all agents or create a new agent.
/api/applications/auto-approve/ POST Auto-approve agent application and create agent account.
